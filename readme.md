* **槪要**  
  * 自販機と**IoT技術**を合わせて手軽に販売データの収集及び分析、  
  そして補充スタッフの便利性まで考えた  
  **リアルタイム自販機統合管理サービス**
  <br>
* **企画の動機**  
  * 日本の多くの自販機
  * 売り切れ自販機
  * **現在サービスの問題点**
    * 自販機の在庫量の把握及び管理の大変さ
    * 販売データの収集の面倒
    * 補充スタッフ1人当たりの自販機の数は７０～１３０台で点検することが大変
<hr>

### **※ 現在サービスの問題点は福岡にある自販機管理会社, 3社に訪問し、不便なところを聞いた問題点です。**

<div>
<img src="https://user-images.githubusercontent.com/37164351/44338995-b3aafd00-a4ba-11e8-829d-421eece7c12e.png">
</div>
<hr>
<br><br><br>

* **主なサービス**
  * 管理者 - ウェブ
    * 会社で管理されている自販機の現在の在庫及び残高の現況を知ることができる**リアルタイム在庫管理サービス**
    * 販売データを分析して点検するべき自販機をお知らせする**販売分析サービス**
    * オペレーターの作業指示書を手軽に生成してくれる**作業指示書生成サービス**
  * 補充スタッフ - モバイル
    * 専用アプリを通じて作業指示書を確認
    * 車両に作業指示の量を積載
    * アプリで補充自販機の位置確認
    * 最寄りの自販機の経路をお知らせ
    * 自販機の飲み物の補充
